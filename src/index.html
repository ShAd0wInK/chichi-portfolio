ClipboardZanager

"ClipboardZanager is a project made by Etienne BAUDOUX. It’s a clipboard manager, which means it allows you to keep an history of all the things you’ve copied. Copied stuffs can be synchronized with the cloud, so you can retrieve those from another device."

1. Speech

I’ve been collaborating on this project for one month only.

The main functionality I added was a setting that allows the user whether passwords and credit card numbers should be synchronized with the cloud (everything is synced by default, except passwords and credit card numbers (thanks to this new feature)).

It took me a bit of time to figure out how everything was linked, and where I had to put my piece of code. Once this was done, it was really nice to see how a big project is structured.

I’ve also worked on another feature, which was the default icon when the app’s icon (from which the data is copied) couldn’t be found (for whatever reason).

This project was the first one where I had the opportunity to work in group using tools such has versioning ones (Github). It took me away from my habit of working solo.
